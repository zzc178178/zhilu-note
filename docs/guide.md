# 📓 知录 · 使用指南

## 什么是知录？

知录是一款轻量级笔记管理 PWA 应用，专为浏览和管理 [ai-note](https://github.com) 生成的 HTML 双页签笔记而设计。支持安装到桌面、离线阅读、文件夹管理、笔记搜索等功能。

## 快速开始

### 在线使用
打开以下链接（建议用 Chrome 或 Safari）：
```
https://zzc178178.github.io/zhilu-note/zhilu.html
```

### 安装到桌面（PWA）

| 设备 | 操作 |
|---|---|
| **Android Chrome** | 地址栏弹出"安装"横幅 → 安装 |
| **iPhone Safari** | 分享按钮 → 添加到主屏幕 |
| **桌面 Chrome/Edge** | 地址栏右侧 ⊕ 图标 → 安装 |

安装后全屏运行，无浏览器地址栏，断网也能使用。

## 核心功能

### 侧边栏操作

| 按钮 | 功能 | 说明 |
|---|---|---|
| ＋ 新建 | 新建文件夹 | 创建文件夹归类笔记 |
| ↑ 导入 | 导入笔记 | 选择本地 HTML 文件批量导入 |
| ↺ 同步 | 智能同步 | 从本地文件夹导入，自动去重、保留层级 |
| ↓ 导出 | 导出备份 | 所有笔记导出为 HTML 文件 |
| ↻ 刷新 | 刷新列表 | 重新加载目录 |

### 笔记操作

- **查看**：点击侧边栏笔记名称预览
- **导入**：拖拽 HTML 文件到页面，或点"导入"按钮
- **编辑**：点 ✏️ 进入所见即所得编辑，也可切源码编辑
- **重命名**：点 📝 按钮
- **导出**：单篇导出或全量导出
- **删除**：点 🗑️ 按钮

### 编辑器

**可视化编辑**：支持加粗（B）、斜体（I）、下划线（U）、标题（H2/H3）、正文（P）、有序/无序列表。

**源码编辑**：直接编辑 HTML 源码，适合精细调整。

### 搜索

侧边栏顶部搜索框，实时筛选笔记名称。

## 移动端使用

- 空页面顶部有 **☰ 打开目录** 按钮
- 打开笔记后，工具栏左侧有 **☰ 按钮**
- 侧边栏从左侧平滑滑入，带半透遮罩
- 点击遮罩、✕ 按钮或选择笔记后自动关闭
- 支持文件选择导入

## 文件导入

### 支持的格式

- `.html` / `.htm`

### 导入方式

1. **选择文件**：侧边栏 → ↑ 导入按钮
2. **拖拽导入**：拖拽到页面中央
3. **手机端**：空页面 → 📁 选择文件导入

### 智能同步

导入目录时自动保留文件夹层级，同名笔记自动去重。支持链接到源文件，文件变化后笔记自动更新。

## 常见问题

### Q: 数据安全吗？
数据存储在浏览器 IndexedDB 中，不经过任何服务器。清除浏览器数据会丢失笔记，建议定期导出备份。

### Q: 怎么备份？
侧边栏 → ↓ 导出，所有笔记导出为 HTML 文件，可随时重新导入。

### Q: 想换浏览器/设备？
导出笔记文件 → 在新设备导入即可迁移。

### Q: 手机看不到侧边栏？
点击 ☰ 按钮或"打开目录"按钮即可展开。

### Q: 图标为什么是"知"字？
取自"知录"之名——记录知识，知晓天下。

## 技术栈

| 技术 | 用途 |
|---|---|
| 原生 HTML/CSS/JS | 单页应用，无框架依赖 |
| IndexedDB | 本地数据存储 |
| PWA / Service Worker | 离线缓存 + 桌面安装 |
| ContentEditable | 可视化编辑 |
| GitHub Pages | 在线部署 |

## 与 ai-note 配合使用

1. 用 ai-note 技能生成双页签 HTML 笔记
2. 在知录中导入 HTML 文件
3. 随时随地离线阅读和管理笔记
